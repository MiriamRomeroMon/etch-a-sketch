# Etch-A-Sketch

A browser-based drawing toy that combines a sketchpad and an Etch-A-Sketch.

## Features

- 16x16 grid created dynamically with JavaScript
- Hover effect that paints squares as you move the mouse
- Button to change the grid size (up to 100x100)
- Random RGB colors on each interaction
- Progressive darkening effect: each square darkens 10% per interaction until fully opaque

## Technologies

- HTML
- CSS (Flexbox)
- JavaScript (DOM manipulation)

## How to use

1. Move the mouse over the grid to draw
2. Click the **NewBlock** button to change the grid size
3. Enter a number between 1 and 100 and a new grid will be generated
